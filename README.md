## Project Description
This is the final, polished version of my personal portfolio web application. It integrates all core web development concepts covered in this course, including responsive design, dynamic JavaScript interactivity, API integration, and performance optimization.

## Key Features & Innovation
- **Innovation Feature:** Implemented a dynamic JavaScript "Typewriter" effect in the header for a professional, engaging user experience.
- **API Integration:** Fetches and displays live GitHub repositories using asynchronous JavaScript (`fetch`).
- **State Management:** User preferences for Dark/Light mode are saved locally using `localStorage`.
- **Complex Logic:** Advanced Regex email validation on the contact form prevents invalid submissions.
- **Performance & Polish:** Deployed globally via GitHub Pages with deferred script loading and clean, structured code.
